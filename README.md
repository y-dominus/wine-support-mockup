# ワイサポ（Wine Support System）モックアップ

ワイン畑の妖精 **Sommia** がサポートするワインマネジメントシステムのモックアップ版です。

![Sommia](sommia.png)

## 🚀 GitHub Pages で公開中

このプロジェクトは GitHub Pages で公開されています。

### 🔗 アクセス方法

1. **トップページ**: `https://[ユーザー名].github.io/wine-support-mockup/`
2. **ダッシュボード**: `https://[ユーザー名].github.io/wine-support-mockup/html/dashboard.html`

## 📋 主な機能

### 🏠 ダッシュボード
- 売上統計とKPI表示
- AIからの在庫アラート
- 人気ワインランキング
- フローティングAIチャット

### 🍷 ワイン管理
- **マイセラー**: 在庫管理とワイン詳細
- **商品リスト**: ワイン検索と詳細表示
- **ワイン最適化**: AI提案による売上向上

### 📦 注文管理
- **補充発注**: 在庫アラートからの自動発注
- **注文履歴**: 過去の発注記録確認

### 📊 レポート・分析
- **売上履歴**: 詳細な売上分析
- **売上レポート**: 月次・年次レポート

### 👤 アカウント管理
- **アカウント設定**: ユーザー情報管理
- **請求管理**: 支払い情報・請求書確認

### 🎯 オンボーディング
- **段階的セットアップ**: 新規ユーザー向けガイド
- **AI案内**: Sommiaによる設定サポート

## 🎨 デザイン特徴

## 🎯 主要機能の使い方

### 1. 初回利用の場合
1. トップページで「はじめて利用する」をクリック
2. Sommiaの案内に従ってセットアップ
3. 店舗情報とワイン要件を入力
4. AIがワインメニューを提案

### 2. 既存ユーザーの場合
1. 「ログイン」からダッシュボードへ
2. 在庫アラートや売上統計を確認
3. フローティングチャットでSommiaに質問

### 3. ワイン管理
- **マイセラー**: 在庫の確認と売上入力
- **商品リスト**: 新しいワインの検索と詳細確認

### 4. AI機能
- **在庫アラート**: 自動的な補充タイミング通知
- **ワイン最適化**: 売上データに基づく改善提案
- **チャットサポート**: 24時間いつでも質問可能

## 🔧 開発・カスタマイズ

### ローカル開発
```bash
# 静的サーバーを起動（Python使用例）
python -m http.server 8000

# ブラウザでアクセス
http://localhost:8000
```

### GitHub Pages への デプロイ
1. GitHubリポジトリを作成
2. ファイルをプッシュ
3. Settings > Pages で公開設定
4. 自動的にURL生成
